# Film-Blog App

## Getting Started

1. Clone the repo:
   ```bash
   git clone <repo-url>
   cd <repo-folder>
````

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start both frontend and backend:

   ```bash
   npm run dev
   ```

The React app will be on [http://localhost:5173](http://localhost:5173)
The API server will be on [http://localhost:3001](http://localhost:3001)
